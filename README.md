# Basic_Pricing_Table
This is a pricing table from Angela Yu's Full Stack Web Development Bootcamp. This project contains 3 divs, with prices and their features. Please note that this is just a simple project and is not used for commercial reasons. This project is responsive and the divs stack when the window is made smaller. 

### What My Application Does
- This inspired other people to code and make projects like this or resemble this
- This shows a sample pricing table which is responsive and adjusts depending on the window size to suit the user's needs
- This shows the prices with different sizes to highlight important information

### Technologies Used
- HTML 5
- CSS 3
- Flexbox
- Media Queries

### Challenges Faced
- Making the media query
- Remembering the Flexbox sizing properties
- Remembering the Flexbox Layout Properties
